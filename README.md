# Mangalam HDPE Pipes - Premium Piping Solutions Website

## Overview

This is a responsive website for Mangalam HDPE Pipes, showcasing premium HDPE pipes and coils for modern infrastructure. The site highlights product features, technical specifications, manufacturing processes, and client testimonials.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Hero Section**: Interactive image gallery with product details and pricing
- **Technical Specs**: Comprehensive specifications table
- **Manufacturing Process**: Step-by-step process visualization with tabs
- **Portfolio**: Product offerings with images and descriptions
- **Testimonials**: Customer reviews in a carousel format
- **FAQ Section**: Expandable frequently asked questions
- **Contact Forms**: Request quotes and download brochures
- **Client Logos**: Display of trusted company partners

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styling with CSS Grid and Flexbox
- **JavaScript**: Interactive elements (gallery, tabs, FAQ, modals)
- **Lucide Icons**: Icon library for UI elements
- **Google Fonts**: Outfit font family

## Installation & Setup

1. **Clone or Download**: Download the project files to your local machine.

2. **File Structure**:

   ```
   project-root/
   ├── index.html          # Main HTML file
   ├── style.css           # Stylesheet
   ├── script.js           # JavaScript functionality
   ├── assists/            # Images and assets
   │   ├── mainLogo.png
   │   ├── BIS-Hallmark-Color.png
   │   ├── ISI-Mark-Black.svg
   │   ├── CE Certification.png
   │   ├── allcomapny.png
   │   ├── man.png
   │   ├── fitting.png
   │   └── Noise & Texture.png
   ├── 14/                 # Additional folders
   └── vercel_out.txt      # Deployment output
   ```

3. **Run the Website**:
   - Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
   - No server required - it's a static website.

## Usage

- **Navigation**: Use the navbar to jump to different sections.
- **Gallery**: Click thumbnails or use arrow buttons to navigate product images.
- **Process Tabs**: Click on manufacturing process steps to view details.
- **FAQ**: Click on questions to expand answers.
- **Forms**: Fill out contact forms for quotes or brochure downloads.
